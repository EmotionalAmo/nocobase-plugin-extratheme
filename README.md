# ExtraTheme（主题增强）

Enhance NocoBase theming beyond the native theme editor (antd tokens only):
custom background (image upload / URL / gradient / color), transparency,
frosted-glass (`backdrop-filter`), and top/side navigation style — across
`/admin` (legacy), `/v` (modern) and the sign-in page. Configured globally in
the plugin settings page. **Disabled by default** — enabling the plugin does
not change NocoBase's appearance until an admin turns it on.

> Package: `@amo/plugin-extratheme` · Settings: `/admin/settings/extra-theme`
